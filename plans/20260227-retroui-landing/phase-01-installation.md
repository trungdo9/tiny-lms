# Phase 01: Installation

## Context

The first phase involves installing RetroUI components and configuring the theme. RetroUI is a NeoBrutalism-styled UI library that integrates with shadcn.

## Overview

Install the RetroUI Button component and configure the theme with the specified yellow primary color and bold shadow styles.

## Key Insights

- RetroUI is distributed as shadcn components via remote URL
- The install command: `npx shadcn@latest add 'https://retroui.dev/r/button.json'`
- RetroUI uses CSS variables for theming, similar to shadcn

## Requirements

1. Run shadcn CLI to add RetroUI Button
2. Add CSS variables for NeoBrutalism theme
3. Configure fonts in layout.tsx

## Architecture

```
frontend/components/ui/
└── button.tsx    # New RetroUI Button (generated by shadcn)
```

## Related Files

- `/home/trung/workspace/project/private/tiny-lms/frontend/app/globals.css` - Add theme variables

## Implementation Steps

### Step 1: Install RetroUI Button

```bash
cd frontend
npx shadcn@latest add 'https://retroui.dev/r/button.json'
```

This will create `/home/trung/workspace/project/private/tiny-lms/frontend/components/ui/button.tsx`.

### Step 2: Update globals.css

Add RetroUI theme variables to `/home/trung/workspace/project/private/tiny-lms/frontend/app/globals.css`:

```css
/* RetroUI NeoBrutalism Theme Variables */
:root {
  /* Primary color - RetroUI yellow */
  --primary: #ffdb33;
  --primary-foreground: #1a1a1a;

  /* NeoBrutalism shadows */
  --shadow-brutal: 4px 4px 0px 0px #000000;
  --shadow-brutal-sm: 2px 2px 0px 0px #000000;
  --shadow-brutal-lg: 6px 6px 0px 0px #000000;

  /* NeoBrutalism borders */
  --border-brutal: 3px solid #000000;

  /* Additional NeoBrutalism colors */
  --bg-primary: #ffdb33;
  --bg-secondary: #ffffff;
  --text-primary: #000000;
}
```

### Step 3: Update layout.tsx fonts

Modify `/home/trung/workspace/project/private/tiny-lms/frontend/app/layout.tsx`:

```tsx
import { Archivo_Black, Space_Grotesk } from "next/font/google";

const archivoBlack = Archivo_Black({
  variable: "--font-archivo-black",
  weight: "400",
  subsets: ["latin"],
});

const spaceGrotesk = Space_Grotesk({
  variable: "--font-space-grotesk",
  subsets: ["latin"],
});

// In body className:
className={`${archivoBlack.variable} ${spaceGrotesk.variable} antialiased`}
```

### Step 4: Add font CSS variables to globals.css

Add to `:root` in globals.css:

```css
--font-archivo-black: var(--font-archivo-black);
--font-space-grotesk: var(--font-space-grotesk);
```
